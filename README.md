# Simple Interest Calculator

A simple command-line Python program that calculates **Simple Interest** and
**Total Amount** based on Principal, Rate of Interest, and Time.

## 📐 Formula

Simple Interest (SI) is calculated as:

```
SI = (P × R × T) / 100
```

Where:
- **P** = Principal amount (the initial sum of money)
- **R** = Rate of interest per year (in %)
- **T** = Time period (in years)

Total Amount is calculated as:

```
Total Amount = P + SI
```

## 🚀 How to Run

1. Make sure you have Python 3 installed.
2. Clone this repository:
   ```bash
   git clone <your-repo-url>
   cd simple-interest-calculator
   ```
3. Run the script:
   ```bash
   python simple_interest_calculator.py
   ```
4. Enter the Principal, Rate of Interest, and Time when prompted.

## 📋 Example

```
=== Simple Interest Calculator ===
Enter Principal amount: 1000
Enter Rate of Interest (%): 5
Enter Time (in years): 2

--- Results ---
Principal Amount : 1000.00
Rate of Interest : 5.00%
Time             : 2.00 years
Simple Interest  : 100.00
Total Amount     : 1100.00
```
